# Security Policy

## Supported Versions

The following version of the pygame-gridcalculator are supported for security patches and updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.0.0   | :x:                |

## General checking

This project has been configured with dependency scanning and code security checking, which executes weekly.  Whilst this project is
at reasonably low risk for any security concerns, if any risks are highlighted via scans or user reports they will be addressed as a
priority.

## Reporting a Vulnerability

If you have identified a vulnerability within this project, please 
[raise a bug](https://github.com/davethepunkyone/pygame-gridcalculator/issues/new?assignees=davethepunkyone&labels=bug&template=bug_report.md&title=%5BBUG%5D)
and I'll investigate as a matter of priority.  Any security updates will be completed as patch releases to be resolved as quickly as possible.
